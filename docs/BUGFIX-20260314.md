## 🐛 Bug #1: 用量页面白屏

### 问题描述
点击"📈 用量"标签后页面白屏，控制台报错：
```
Uncaught ReferenceError: UsageMonitor is not defined
at App (App.jsx:171:16)
```

### 根本原因
在 `App.jsx` 中：
1. 导入的组件名是 `UsageMonitorEnhanced`
2. 但在渲染时使用了 `UsageMonitor`（旧名称）
3. 导致 ReferenceError

**错误代码**:
```jsx
import UsageMonitorEnhanced from './components/UsageMonitorEnhanced';

// ❌ 错误：使用了未定义的组件名
{activeTab === 'usage' && (
  <UsageMonitor />
)}

{activeTab === 'usage' && (
  <UsageMonitorEnhanced />  // 重复的条件
)}
```

### 修复方案
删除重复的条件渲染，使用正确的组件名：

**修复后代码**:
```jsx
{activeTab === 'usage' && (
  <UsageMonitorEnhanced />
)}
```

- ✅ 实时用量卡片显示
- ✅ 用量趋势图正常
- ✅ 模型排行正常

## 🐛 Bug #2: Skills 分类筛选要点两次

### 问题描述
在 Skills 管理页面点击分类标签时，需要点击两次才能切换到对应分类。

### 根本原因
在 `SkillsManager.jsx` 中：
1. `handleCategoryChange` 函数先调用 `setSelectedCategory(category)`
2. 然后用 `setTimeout(() => loadSkills(), 100)` 加载数据
3. 但 `loadSkills()` 读取的是 `selectedCategory` 状态（旧值）
4. React 状态更新是异步的，第一次点击时状态还未更新

**错误代码**:
```jsx
const handleCategoryChange = (category) => {
  setSelectedCategory(category);  // 异步更新
  setLoading(true);
  setTimeout(() => loadSkills(), 100);  // loadSkills 读取旧状态
};

const loadSkills = async () => {
  // 使用 selectedCategory（可能是旧值）
  if (selectedCategory !== 'all') params.append('category', selectedCategory);
  // ...
};
```

### 修复方案
创建新函数 `loadSkillsWithCategory`，直接使用传入的 category 参数：

**修复后代码**:
```jsx
const handleCategoryChange = (category) => {
  setSelectedCategory(category);
  setLoading(true);
  // 直接加载，使用传入的 category 参数
  loadSkillsWithCategory(category);
};

const loadSkillsWithCategory = async (category) => {
  try {
    const params = new URLSearchParams();
    if (category !== 'all') params.append('category', category);  // 使用参数而非状态
    if (selectedSource !== 'all') params.append('source', selectedSource);
    if (searchQuery) params.append('search', searchQuery);

    const res = await fetch(`http://localhost:3001/api/skills/list?${params}`);
    const result = await res.json();
    if (result.success) {
      setSkills(result.skills);
    }
  } catch (error) {
    console.error('加载 Skills 失败:', error);
  } finally {
    setLoading(false);
  }
};
```
