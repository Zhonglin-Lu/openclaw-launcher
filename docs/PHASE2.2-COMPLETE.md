## 🎯 核心功能详情

### 1. 诊断中心

**诊断流程**:
```
用户点击"开始诊断"
  ↓
调用 /api/diagnose
  ↓
执行 diagnostician.js
  ↓
8 项检查:
  - OpenClaw 安装
  - 配置文件
  - Workspace
  - Gateway 状态
  - Node.js 版本
  - 文件权限
  - 网络连接
  - Skills 状态
  ↓
生成结构化报告
  ↓
显示可视化结果
  ↓
提供一键修复
```

**修复流程**:
```
用户点击"一键修复"
  ↓
显示"修复中..."
  ↓
执行修复命令
  ↓
记录修复历史
  ↓
重新诊断验证
  ↓
显示修复结果
```
